üìù How to Add New Releases

To add a new release, edit the index.html file and add a new article block:

  1. Copy one of the existing release article blocks
  2. Update the version number and release date
  3. Change the release type badge (major, minor, or patch)
  4. Fill in your release notes content
  5. Commit and push your changes to GitHub
  6. GitHub Pages will automatically update within a few minutes

Release Type Guidelines:

  • MAJOR: Breaking changes or significant new features
  • MINOR: New features without breaking changes
  • PATCH: Bug fixes and minor improvements
